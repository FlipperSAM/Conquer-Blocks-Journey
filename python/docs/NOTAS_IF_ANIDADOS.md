# 🔀 Lógica de Condicionales Anidados

En este bloque profundizo en estructuras de decisión complejas donde el resultado de una evaluación depende de una validación previa.

### Puntos Clave:
*   **Filtros de Seguridad:** En el proyecto de la **Renta**, se aplica un "filtro de entrada" (edad e ingresos) antes de procesar el cálculo. Si no se cumple, el programa ahorra recursos no ejecutando el resto de la lógica.
*   **Selección Dinámica:** En la **Hamburguesería**, el programa adapta sus respuestas según la entrada del usuario, simulando una interfaz de usuario real.
*   **Normalización de Entrada:** Uso de `.lower()` y `.title()` para asegurar que la comparación no falle por mayúsculas accidentales.
