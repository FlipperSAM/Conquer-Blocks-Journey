# Lógica de Negocio: Cálculo de Comisiones

En este proyecto se implementa un sistema de cálculo de incentivos basado en diferentes categorías de productos, una tarea común en el desarrollo de software administrativo.

### Conceptos Clave:
*   **Acumuladores:** Uso de la variable `total_comision_mes` para sumar los resultados de diferentes iteraciones.
*   **Listas Paralelas:** Organización de datos (Nombres, Precios, Comisiones) para ser procesados mediante un índice común.
*   **Formateo de Moneda Profesional:** Uso de `:, .2f` para incluir separadores de miles y dos decimales, asegurando que los montos altos sean legibles.

> **Importancia Profesional:** Demostrar control sobre diferentes porcentajes de comisión según el tipo de producto evita errores contables en aplicaciones reales.
