# Lógica de Programación Aplicada al Trading

Este proyecto simula la toma de decisiones de un bot financiero basado en umbrales de precio predefinidos.

### Aplicación de Condicionales:
*   **Segmentación de Precios:** Uso de `elif` para gestionar el rango intermedio (100-150), asegurando que solo se ejecute una orden a la vez.
*   **Lógica de Inversión:** Implementación de reglas de mercado básicas (Buy Low, Sell High).
*   **Validación de Datos:** Uso de `float` para permitir precios con centavos, crucial en mercados financieros.

> **Habilidad Tech:** Los sistemas de decisión automatizada son la base de los algoritmos de alta frecuencia en el sector bancario y de criptoactivos en España.
