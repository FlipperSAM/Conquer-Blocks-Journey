# Seguridad de Datos: Enmascaramiento de Strings

En el desarrollo de software financiero, la **privacidad de los datos** es una prioridad absoluta. Este ejercicio demuestra cómo proteger información sensible mediante técnicas de manipulación de cadenas.

### Conceptos Clave:
*   **Slicing `[-4:]`**: Permite extraer los últimos 4 caracteres de un string sin importar su longitud total.
*   **Operador de Repetición `*`**: Usado para generar dinámicamente la cantidad de asteriscos necesarios según la longitud de la entrada.
*   **Data Masking**: Es el nombre técnico de este proceso, fundamental para cumplir con normativas como la **GDPR** en España y Europa.

> **Importancia Profesional:** Mostrar que comprendes la necesidad de ocultar datos sensibles en pantalla demuestra madurez técnica y responsabilidad como desarrollador.
