# 📚 Estructuras de Datos: Diccionarios en Python (restaurante_cuenta.py)

En el proyecto del **Restaurante**, he evolucionado el uso de listas hacia **Diccionarios (`dict`)**, una herramienta fundamental para el mapeo de datos.

### ¿Por qué usar Diccionarios?
*   **Mapeo Clave-Valor:** Permite asociar directamente el nombre del producto (clave) con su precio (valor).
*   **Iteración Eficiente:** Con el método `.items()`, podemos recorrer fácilmente nombres y precios al mismo tiempo.
*   **Escalabilidad:** Añadir un nuevo plato al menú es tan simple como agregar una línea al diccionario, sin cambiar la lógica del programa.

> **Dato Pro:** En el desarrollo de software en España, el uso de diccionarios es la base para manejar datos que vienen de bases de datos o APIs en formato JSON.
