# Listas en Python: Manipulación de Colecciones

Las listas son una de las estructuras más potentes de Python. A diferencia de las variables simples, permiten almacenar múltiples elementos en un solo lugar.

### Conceptos Fundamentales:
*   **Índices:** El acceso siempre comienza en **0**. Si una lista tiene 4 elementos, los índices son 0, 1, 2 y 3.
*   **Mutabilidad:** He practicado la modificación de elementos mediante el acceso directo por índice (`lista[0] = nuevo_valor`).
*   **Gestión Dinámica:**
    *   `.append()`: Ideal para flujos donde los datos llegan uno tras otro.
    *   `.insert(pos, valor)`: Útil cuando el orden de los datos es crítico.
    *   `.remove(valor)`: Permite limpiar la colección buscando un elemento específico.

> **Importancia en el mercado:** Saber gestionar colecciones de datos es indispensable para trabajar con APIs de comercio electrónico y gestión de usuarios en el backend.
