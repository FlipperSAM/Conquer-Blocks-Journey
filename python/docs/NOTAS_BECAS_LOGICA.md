# 📚 Lógica de Validación: Becas de Excelencia

En este proyecto implemento un sistema de filtros múltiples utilizando **operadores lógicos**, una técnica crítica para procesos de selección automatizados.

### Conceptos Clave:
*   **Operador `and`**: Se utiliza para asegurar que el estudiante cumpla **ambos** criterios (excelencia académica y rango de edad) de forma obligatoria.
*   **Encadenamiento de Comparaciones**: Uso de la sintaxis `17 <= edad <= 21` para validar rangos de forma limpia y eficiente.
*   **Programación Defensiva**: Implementación de `try-except` para gestionar errores de entrada de datos, asegurando que el programa no se detenga inesperadamente.
