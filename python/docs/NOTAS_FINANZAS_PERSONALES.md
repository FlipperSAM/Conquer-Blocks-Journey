# 📊 Proyecto: Análisis de Ahorro y Escenarios (Calculadora_ahorros)

En este proyecto integrador se aplican todos los conocimientos de aritmética y lógica de control para resolver un problema de finanzas personales.

### Lógica Aplicada:
*   **Proyección Anual:** Multiplicación de variables semanales por la constante de tiempo (52 semanas).
*   **Análisis de Sensibilidad:** Modificación de variables (horas y gastos) para evaluar la viabilidad financiera bajo condiciones de "tiempo parcial".
*   **Formateo de Moneda:** Uso de `:, .2f` para manejar montos que pueden superar los mil euros, mejorando la legibilidad.

> **Habilidad Técnica:** La capacidad de crear modelos que respondan a preguntas de tipo "Qué pasaría si..." (What-if analysis) es muy valorada en el desarrollo de software de gestión en España.
