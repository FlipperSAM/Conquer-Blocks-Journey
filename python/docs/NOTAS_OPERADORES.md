# Operadores Aritméticos y Tipos de Datos

En este ejercicio de la **Casa de Cambios**, he aplicado conceptos fundamentales para el manejo de datos numéricos en Python.

### Conceptos Clave:
*   **`float()`**: Conversión de la entrada (string) a número decimal. Es indispensable para cálculos financieros.
*   **Operadores**: Uso de `*` (multiplicación) para la tasa y `-` (resta) para aplicar la comisión.
*   **Formateo `: .2f`**: Técnica para limitar los decimales a dos posiciones, estándar internacional para representar dinero.
*   **Constantes**: Definición de `TASA_CAMBIO` en mayúsculas para seguir las convenciones de *Clean Code*.

> **Nota para Reclutadores:** El manejo preciso de tipos de datos es la base para construir aplicaciones seguras y escalables en entornos de producción.
